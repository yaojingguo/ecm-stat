Protocol

The following text uses an example to describe protocol between client and server. Server end receives a request:

{beginDate: "2011-01", endDate: ""2011-05"", "chartType": 0}. It produces the following result: 
  {"x": ["2011-01", "2011-02", "2011-03", "2011-04", "2011-05"]
   "y": [23, 26, 100, 45, 50]
   "chartType": 0
  }
The format of date decides the report type:
  1. yyyy-MM: month report
  2. yyyy-MM-dd: day report
  3. yyyy-MM-dd HH: hour report

MM begins with 1. dd begins with 1. HH begins with 0. For a hour report for one day, an example is from 2011-01-01 00 to2011-01-01 23 .
